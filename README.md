<h3>课程介绍：</h3>
<p>Laravel 默认的目录结构试图为不管是大型应用还是小型应用提供一个良好的起点。当然，你也可以按照自己的喜好重新组织应用的目录结构，因为 Laravel 对于指定类在何处被加载没有任何限制 —— 只要 Composer 可以自动载入它们即可。</p>
<p>Laravel 5.8 在美国时间2019年2月26号正式发布了，该版本引入了一些新特性以及最新的bug修复，并对框架核心代码做了优化。</p>
